    Converte a magnitude do sinal para escala logarítmica (dB), seguindo a expressão:

    \[ Signal_{DB} = 20 \cdot \log_{10}(|signal| + 10^{-12}) \]
    Nota

    O termo \(10^{-12}\) é adicionado para evitar logaritmo de zero, que é indefinido.
