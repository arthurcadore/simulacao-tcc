    Convolutional Encoder

    __init__(G=np.array([[121, 91]]))

    Inicializa o codificador convolucional com matriz de geradores.

    Referência
    • AS3-SP-516-274-CNES (seção 3.1.4.4)
    • CCSDS 131.1-G-2

    Parameters:

    Name Type Description Default
    G ndarray

    Matriz de polinômios geradores em formato binário.

    		 array([[121, 91]])
    Nota
    • O polinômio gerador \(G_0\) é representado por G[0][0] e \(G_1\) por G[0][1].
    • \(G_0\) é definido como \(1111001_{2}\) ou \(121_{10}\)
    • \(G_1\) é definido como \(1011011_{2}\) ou \(91_{10}\)

    calc_free_distance()

    Calcula a distância livre do código convolucional, definida como a menor distância de Hamming entre quaisquer duas sequências de saída distintas.

    Returns:

    Name Type Description
    dist int

    Distância livre do código.
